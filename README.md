# owlos-template-sc

A clean, professional **language school website template** — single page, mobile-friendly,
100% self-contained, GDPR compliant out of the box.

Built around a Slovak language school concept (A1–C2 courses for foreigners),
easily adapted to any language or tutoring business.

> **Template demo** — replace all placeholder content (school name, courses, prices,
> teachers, contact details) before deploying as a real website.

---

## Features

- **Single page** — Hero, About, Courses (A1–C2), Pricing, Teachers, Testimonials, FAQ, Contact
- **Clean blue / red design** — system-ui fonts only; zero font CDN requests
- **Zero external dependencies** — no Google Fonts, no CDN, no analytics, no tracking
- **GDPR compliant by default** — no external HTTP requests at page load whatsoever
- **Animated canvas background** — lightweight, CPU-friendly particle effect
- **Demo disclaimer banner** — fixed-position notice with localStorage dismissal;
  remove the banner block before going live
- **MIT License** — use freely, replace `[Your Company Name]` in `LICENSE`

---

## Quick Start

1. Download or clone this repository
2. Open `index.html` in your browser — no build step required
3. Replace placeholder content:
   - School name (search for `SC Slovenčina`)
   - Course names, descriptions, and prices
   - Teacher names, photos, and bios
   - Phone, email, address (search for `+421 900 000 000` / `info@sc-slovencina.sk`)
   - Loader tagline — replace the `<!-- TODO -->` text with your school's slogan
   - Replace `© 2026 Your Company Name` in the footer
   - Remove the `<!-- Demo disclaimer -->` block before going live
4. Deploy: upload `index.html` to any static web host — no other files required

## GDPR Notice

This template is GDPR-friendly by design:

- **Fonts** — system-ui stack only (OS fonts, no network request)
- **No external scripts** — no CDN, no jQuery, no framework loaded remotely
- **No analytics** — no Google Analytics, Matomo, Hotjar, or similar
- **No tracking pixels** — no Facebook Pixel or third-party pixels
- **No contact form backend** — the form does not submit data anywhere in demo mode;
  connect your own backend or email service before going live
